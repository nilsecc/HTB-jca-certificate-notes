Controls access to files and directories (who can read, write or execute)

Each file has:

- **Owner (user)**
- **Group**
- **Others**

---

### Permission Types

**r (read)**  
View file content

**w (write)**  
Modify file or directory (create/delete files)

**x (execute)**  
Run files OR traverse directories

---

### Directory Permissions (IMPORTANT)

**x on directory**  
Needed to enter/traverse it

**r on directory**  
List files inside

**w on directory**  
Create/delete/rename files inside

---

### View Permissions

**ls -l**  
`ls -l /etc/passwd`

Example:

-rwxr-xr-- 1 root root 1641 May 4 23:42 /etc/passwd

Breakdown:

- `-` → file (`d` = directory)
- `rwx` → owner
- `r-x` → group
- `r--` → others

---

### Change Permissions

**chmod (symbolic)**  
`chmod u+x file`  
Add execute to owner

`chmod g-w file`  
Remove write from group

`chmod a+r file`  
Add read to all

---

**chmod (octal)**  
`chmod 754 file`

Meaning:

- 7 → rwx (4+2+1)
- 5 → r-x (4+0+1)
- 4 → r-- (4+0+0)

---

### Octal Reference

|Number|Permission|
|---|---|
|4|r|
|2|w|
|1|x|

Examples:

- 7 = rwx
- 6 = rw-
- 5 = r-x
- 4 = r--

---

### Change Owner

**chown**  
`chown user:group file`

Example:  
`chown root:root shell`

---

### Special Permissions

---

### SUID

**Runs file as owner**  
`chmod u+s file`

Shown as:  
`-rwsr-xr-x`

Used for privilege escalation (dangerous if misconfigured)

---

### SGID

**Runs file as group**  
`chmod g+s file`

Also used on directories → new files inherit group

---

### Sticky Bit

**Only owner can delete files in directory**  
`chmod +t directory`

Example:  
`drwxrwxr-t`

---

### Sticky Bit Cases

**t (lowercase)**  
Execute permission present → normal access

**T (uppercase)**  
No execute → cannot access directory contents

---

### Practical Examples

**Give execute to owner**  
`chmod u+x script.sh`

**Remove write from others**  
`chmod o-w file`

**Full permissions to owner only**  
`chmod 700 file`

**Shared directory with protection**  
`chmod 1777 /shared`

---

### Mental Model

- Files → `x` = execute
- Directories → `x` = enter
- `r` = see
- `w` = modify
- `chmod` → change permissions
- `chown` → change owner
