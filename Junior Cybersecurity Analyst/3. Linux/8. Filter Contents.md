### Paginators

**more**
`more ~/.congif/alacracitty/alacracitty.toml`
Displays the output in a simple text viewer. You can move forward page by page (Space or Enter).When you exit with q, the content remains visible in the terminal.

**less**
`less ~/.congif/alacracitty/alacracitty.toml`
Works like `more` but with better navigation and display features.
Features: Scroll forward and backward, search with `/text`, go to start (`g`) or end (`G`), better handling of large files.

When you exit with `q`, the screen is usually restored to how it was before opening `less`(so the content does not remain visible).

### Filters

**head**
`head /etc/passwd`
Show the 10 first lines of a textfile or output if pipping
`head -n 20 /etc/passwd`
Now 20 first lines

**tail**
`tail /etc/passwd`
Same but from the end
`tail -f /logs.txt`
Useful to see dynamically the new last lines of a file being modyfied like real time logs

**sort**
`cat /etc/passwd | sort`
Sort the output

**grep**
`cat /passwords.txt | sort "root"`
Pattern searching
`cat /passwords.txt | sort -v " "`
To ignore Patterns

**cut**
`cat /etc/passwd | grep -v "false\|nologin" | cut -d":" -f1`
Display a column separated from a delimeter

**tr**
`cat /etc/passwd | grep -v "false\|nologin" | tr ":" " "`
To replace a character to another

**column**
```
cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | column -t

root         x  0     0      root               /root        		 /bin/bash
sync         x  4     65534  sync               /bin         		 /bin/sync
postgres     x  111   117    PostgreSQL         administrator,,,    /var/lib/postgresql		/bin/bash
mrb3n        x  1000  1000   mrb3n              /home/mrb3n  	     /bin/bash
cry0l1t3     x  1001  1001   /home/cry0l1t3     /bin/bash
htb-student  x  1002  1002   /home/htb-student  /bin/bash
```
Separate columns better with tabs

**awk**
```
cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | awk '{print $1, $NF}'

root /bin/bash
sync /bin/sync
postgres /bin/bash
mrb3n /bin/bash
cry0l1t3 /bin/bash
htb-student /bin/bash
```
display first and last column

**sed**
```shell-session
cat /etc/passwd | grep -v "false\|nologin" | tr ":" " " | awk '{print $1, $NF}' | sed 's/bin/HTB/g'

root /HTB/bash
sync /HTB/sync
postgres /HTB/bash
mrb3n /HTB/bash
cry0l1t3 /HTB/bash
htb-student /HTB/bash
```
Replace patterns in this case replaces "bin" with "HTB"

**wc**
`cat /users.txt | wc -l`
Display the number of, in this case, lines. Useful counter
