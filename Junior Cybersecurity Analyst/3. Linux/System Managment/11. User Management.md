Managing users and groups to control access and permissions in the system

---

### sudo

**Execute command as another user (usually root)**  
`sudo command`

Example:  
`sudo cat /etc/shadow`

Used to perform admin tasks without logging as root

---

### su

**Switch user**  
`su`  
Switch to root (asks password)

`su user`  
Switch to another user

---

### useradd

**Create new user**  
`sudo useradd user`

With home directory:  
`sudo useradd -m user`

---

### userdel

**Delete user**  
`sudo userdel user`

Delete user + home:  
`sudo userdel -r user`

---

### usermod

**Modify user**  
`sudo usermod options user`

Example (add to group):  
`sudo usermod -aG group user`

---

### addgroup

**Create new group**  
`sudo addgroup group`

---

### delgroup

**Delete group**  
`sudo delgroup group`

---

### passwd

**Change password**  
`sudo passwd user`

---

### Important Files

**/etc/passwd**  
`cat /etc/passwd`  
User info (NOT passwords)

**/etc/shadow**  
`cat /etc/shadow`  
Encrypted passwords (root only)
